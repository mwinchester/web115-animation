Exploring CSS Animation

Why use CSS for animation?

The first animations on the web were Animated Gifs. They allowed animation by having multiple images change on the screen, creating an animation in the style of an old flip book. These still work, and you will see them on the web.

The next major player in web animation was Macromedia (later Adobe) Flash. Flash used scalable vector graphics to produce animations. At one time 98% of web accessible devices worked with Flash.

Then the mobile revolution happened, and mobile devices quit supporting Flash. A better way to animate, that would work with web standards, needed to develop.

Now you can create simple animations with CSS3. For more advanced animation techniques, you will need to learn JavaScript.

For a more complete reference, with examples, please see: W3Schools CSS3 Animations

Animating a Div to move

The heart image is inside the div, so by moving the div, we animate the heart. The div has an id="heart" so that we can specify exactly what we want to animate.

The code looks like this:

#heart{
width: 300px;
height: 300px;
position: relative;
left:200px;
-webkit-animation-name: heart-move; /* Chrome, Safari, Opera */
-webkit-animation-duration: 4s; /* Chrome, Safari, Opera */
animation-name: heart-move;
animation-duration: 6s;
}

@keyframes heart-move {
0% {left:200px; top:0px;}
25% {left:500px; top:0px;}
50% {left:500px; top:200px;}
75% {left:200px; top:200px;}
100% {left:200px; top:0px;}
}

You can also change how many times the animation will loop, delay start, colors, etc. Please check out the W3schools site for a full reference.
